## Как запустить у себя

0. Устанавливаем Node.js и npm
1. Клонируем проект
2. Копируем `.env.example` в `.env.local` и изменяем там параметры на свои
3. `$ npm run dev`

## Краткие штуки о разработке

* В `pages/api/custom.js` функции для запросов к серверу.
